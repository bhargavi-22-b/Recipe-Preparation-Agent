# Recipe-Preparation-Agent
AI-powered Recipe Preparation Agent built using IBM watsonx.ai, Meta Llama 3.3 70B Instruct, and Langflow. Generates personalized recipes, step-by-step cooking instructions, ingredient substitutions, and cooking tips based on user-provided ingredients.

Recipe Preparation Agent
IBM University Engagement Project | Food & Nutrition Domain

An AI-powered Recipe Preparation Agent built using 
IBM watsonx.ai, Meta Llama 3.3 70B Instruct, and Langflow.

About the Project
This agent helps users cook meals using only the 
ingredients they have on hand. Users input available 
ingredients and receive personalized recipe suggestions 
with step-by-step cooking instructions, ingredient 
substitutions, cooking tips, and dietary recommendations.

Problem Statement
Many individuals face difficulty deciding what meals 
to prepare using available kitchen ingredients. This 
agent solves that by generating practical recipes, 
reducing food waste, and simplifying meal planning 
using Generative AI.
